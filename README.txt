Instruções de Instalação

Para instalar a plataforma, é necessário apenas um servidor Tomcat configurado e pronto a correr. No nosso caso, utilizámos um servidor Tomcat configurado e integrado com o IntelliJ da Jetbrains.
Depois disso, deve ser executado a função main() da classe TDB de forma a criar a base de dados necessária para a utilização da aplicação.

Instruções de Utilização

Para correr o nosso projecto, basta correr o servidor Tomcat previamente configurado. A partir daí, é aceder ao link do artifacto gerado.
